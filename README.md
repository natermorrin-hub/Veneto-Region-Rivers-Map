# Veneto-Region-Rivers-Map
This reporistory has been created for hosting an interactive map created with the qgis2web plugin. Explore the map here: <a href="https://natermorrin-hub.github.io/Veneto-Region-Rivers-Map/VenetoWaterwaysMap/" target="_blank">Veneto Region Rivers Map</a>

# Map Description
A map of major rivers and other minor waterways in the Veneto region that drain into the Venice Lagoon. Data includes source, extent, and mouth of waterways as well as the boundaries of the Venice Lagoon and the basins of the Veneto Region.

# Layers
- Source (Estimated location of the source of major rivers)
- Mouth (Estimated location of the location in which a major river empties into the Venice Lagoon)
- Major Rivers (Path of major rivers)
- All Waterways (All waterways in the Veneto region)
- Venice Lagoon Boundaries (boundaires of the Venice Lagoon)
- Veneto Region Basin Boundaries (boundaries of all river basins in the Veneto Region)
- Street Layer (OpenStreetMap Street Layer)
- Topography (ESRI World Topographic Map Topography Layer)

# Usage
Within this interactive map you can
- control visibility of the layers (layer menu exists in the right upper corner)
- pan the map (click & drag)
- zoom in or out (plus or minus buttons)
- geolocate yourself (pin button)
- measure distances (ruler button; start the measurement with one click and stop with double click)
- search for addresses or place names (magnifying glass button)
